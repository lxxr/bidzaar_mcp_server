# Навык: Публикация закупочной процедуры на Bidzaar

## Описание

Этот навык позволяет автоматизировать процесс публикации закупочной процедуры на платформе Bidzaar. 
Навык использует MCP-инструмент `create_procedure` для взаимодействия с API Bidzaar.

## Доступные MCP инструменты
- `create_procedure` - создание и публикация процедуры
- `update_procedure` - обновление информации в процедуре
- `get_procedure` - получение информации о процедуре
- `upload_files` - загрузка файлов

## Процесс публикации процедуры

### Шаг 1: Сбор информации о процедуре

Перед созданием процедуры необходимо собрать следующую информацию:

1. **Название процедуры** (обязательно)
2. **Описание** (обязательно) — подробное описание предмета закупки
3. **Контакты** (обязательно) — если не заданы, используй `procurement@3z.ru`
4.1 **Тип процедуры (type)** - для закупок `1`, для продаж `2`, для реестров `3`
4.2 **Тип закупки (tradingType)** (по умолчанию `8` - мониторинг рынка/RFI):
   - `1` — торги за объем (RFP)
   - `2` — торги за единицу (RFQ)
   - `4` — квалификация (ПКО)
   - `8` — мониторинг цен (RFI)
5. **Товары/услуги** — список позиций (для RFI добавляй в описание)
6. **Дата окончания приема заявок** (минимум 7 дней от текущей даты)
7. **Валюта** (по умолчанию RUB)

### Шаг 2: Загрузка файлов (если есть)

Если пользователь прикрепил файлы к сообщению или указал пути к файлам:

1. Сначала загрузи файлы через `upload_files`
2. Для каждого файла используй `file_path` (путь к файлу на сервере)
3. Сохрани полученные ID файлов для добавления в процедуру

Пример вызова `upload_files`:
```json
{
  "files": [
    {
      "name": "техническое_задание.pdf",
      "extension": "pdf",
      "file_path": "/tmp/cyclop/uploads/technical_spec.pdf"
    }
  ]
}
```

Формат ответа

4. При успешной публикации процедуры на Bidzaar, ВСЕГДА При успешной публикации процедуры на Bidzaar, используй форматированный ответ со ссылкой на площадку (https://phoenix.bidzaar.com/process/light/<id процедуры>)
    ✅ Процедура успешно опубликована!\n\n
    
    📋 **Название:** [название процедуры]\n
    🆔 **ID процедуры:** [id]\n
    🔗 **Ссылка:** [ссылка на процедуру (url площадки https://phoenix.bidzaar.com/process/light/ и id процедуры))]\n
    📊 **Статус:** Опубликована\n\n
    🏢 **Приглашенные поставщики:** [количество] компаний\n
    📌 **Теги:** [список тегов]\n\n
    📎 **Документы:** [список загруженных файлов]\n\n

2. Уточни надо ли пригласить поставщиков (нужный формат Название ИНН email)
    
3. Если процедура создана, но не опубликована:
